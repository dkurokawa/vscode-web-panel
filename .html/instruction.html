🌐 VSCode Web Panel

Embed Web Dashboards Directly in VS Code

0.1.1

概要

🎯 拡張機能の概要

VSCode Web Panelは、任意のWebダッシュボードやWebアプリケーションをVS Codeのパネルやサイドビューに埋め込んで表示できる拡張機能です。

開発に必要な情報ダッシュボード、監視ツール、ドキュメントなどを、エディタから離れることなく参照できます。

💻 VS Code 1.102.0+ 🌐 Any Web Content 🎨 Customizable Layout 🔒 Security Features

開発ワークフローを効率化するために、頻繁に参照するWebページやダッシュボードをVS Code内に統合できます。

// コマンドパレットから実行 > Web Panel: Open in Panel > Web Panel: Open in Side View > Web Panel: Set Default URL

🖼️ 柔軟な表示オプション

パネル（下部）またはサイドビュー（横）での表示を選択可能

🏠 デフォルトダッシュボード

URLが指定されていない時に表示されるカスタマイズ可能なホームページ

🔐 セキュリティ機能

CSPとサンドボックス化によるセキュアな実行環境

💾 永続的な設定

URLとレイアウトの設定を保存し、次回起動時に復元

📖 詳細機能

主な機能

  • 2つの表示モード: パネル（ボトムパネル）とサイドビュー（エクスプローラーサイドバー）
  • インタラクティブな設定: 表示中のパネルから直接URLを設定可能
  • デフォルトダッシュボード: カスタマイズ可能なスタートページ
  • セキュリティ強化: iframe sandboxとCSPによる保護
  • 永続化: ワークスペース設定にURLを保存
  • リロード機能: コンテンツの手動リフレッシュ

使用例

  • 📊 プロジェクトのメトリクスダッシュボード表示
  • 📚 APIドキュメントの参照
  • 🔍 ログビューアーの埋め込み
  • 📈 パフォーマンスモニタリング
  • 🗺️ プロジェクトのロードマップ表示
  • 🎨 デザインシステムのリファレンス

セキュリティ機能

Web Panelは以下のセキュリティ対策を実装しています：

  • iframe sandbox属性: スクリプト実行とフォーム送信を許可し、その他は制限
  • Content Security Policy: XSSおよびデータインジェクション攻撃から保護
  • Origin分離: VS Codeの拡張機能APIへのアクセスを防止

インストール方法

🚀 VS Code Marketplace からインストール

  1. VS Code の拡張機能パネルを開く（Cmd/Ctrl + Shift + X）
  2. "Web Panel" を検索
  3. インストールボタンをクリック

📦 VSIX ファイルからインストール

# コマンドラインから
code --install-extension vscode-web-panel-0.1.1.vsix

# または VS Code UI から
1. 拡張機能パネルを開く
2. ... メニューから「VSIXからインストール」を選択
3. VSIXファイルを選択

クイックスタート

1. パネルで開く

1. コマンドパレットを開く (Cmd/Ctrl + Shift + P)
2. "Web Panel: Open in Panel" を実行
3. URLを入力（または空欄でデフォルトダッシュボード）

2. サイドビューで開く

1. コマンドパレットを開く (Cmd/Ctrl + Shift + P)
2. "Web Panel: Open in Side View" を実行
3. エクスプローラーサイドバーに Web Panel が表示される

3. デフォルトURLの設定

// settings.json に追加
{
  "webPanel.defaultUrl": "https://example.com/dashboard"
}
💡 ヒント: デフォルトダッシュボードをカスタマイズするには、拡張機能のソースコードのdashboard-template.htmlを編集してください。

設定オプション

利用可能な設定

  • webPanel.defaultUrl - デフォルトで開くURL
  • webPanel.openInPanel - 起動時に自動的にパネルを開く
  • webPanel.openInSideView - 起動時に自動的にサイドビューを開く

コマンド

  • webPanel.openInPanel - パネルでWebページを開く
  • webPanel.openInSideView - サイドビューでWebページを開く
  • webPanel.setDefaultUrl - デフォルトURLを設定
  • webPanel.reload - 現在のコンテンツをリロード

開発者向け情報

🔧 ローカルでのビルドと実行

# リポジトリをクローン
git clone https://github.com/dkurokawa/vscode-web-panel.git
cd vscode-web-panel

# 依存関係をインストール
npm install

# 拡張機能をビルド
npm run compile

# VS Codeでデバッグ実行
# F5キーを押して新しいVS Codeウィンドウで実行

📝 カスタムダッシュボードの作成

src/dashboard-template.htmlを編集して、独自のデフォルトダッシュボードを作成できます。

// dashboard-template.html の例
<!DOCTYPE html>
<html>
<head>
    <title>My Custom Dashboard</title>
</head>
<body>
    <h1>Project Dashboard</h1>
    <!-- カスタムコンテンツ -->
</body>
</html>
